This is my first javascript game.
It was made in early 2014.

It is a multiplayer game with 2 ninjas.

Here is the controls (azerty keyboard recommended) :

Left player :
    - Z : up
    - Q : left
    - S : down
    - D : right

    - B : shuriken
    - N : sword

Right player :
    - Up    arrow : up
    - Left  arrow : left
    - Down  arrow : down
    - Right arrow : right

    - P : shuriken
    - O : sword
